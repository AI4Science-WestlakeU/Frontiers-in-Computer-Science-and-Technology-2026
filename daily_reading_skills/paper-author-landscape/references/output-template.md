# 输出模板

## 证据等级

- `高`：论文原文、作者主页、实验室主页、机构官方 publication 页面、项目主页直接支持
- `中`：作者近期论文连续共现，且可交叉验证
- `低`：只由关键词、合作关系或单篇相似论文推断

## 推荐输出格式

### 单篇模式

```markdown
## 这篇论文的作者与机构

- 作者 / 机构：
- 这篇工作里的角色：
- 为什么说明他们在做这个方向：

## 这个方向谁在做

- 作者：
  - 姓名 / 当前机构
  - 相关问题
  - 代表论文
  - 证据等级

- 实验室：
  - 实验室 / PI
  - 近期连续工作
  - 为什么值得跟踪

- 机构：
  - 机构名
  - 在这个方向的主要切口
  - 代表组或代表论文

## 值得长期跟踪的组

- 组 / 机构
- 跟踪理由
- 建议继续看的论文或项目

## 边界与不确定性

- 哪些是直接证据
- 哪些是推断
- 哪些相关但没有纳入主名单
```

### 汇总模式

```markdown
## 当前汇总范围

- 纳入的论文：
- 这里的“对应领域”如何界定：
- 没纳入主名单的近邻主题：

## 当前库里的作者与机构概览

- 高频作者：
- 高频机构：
- 明显的合作簇：

## 当前库中重复出现的作者 / 组 / 机构

- 作者 / 机构
- 在哪几篇里出现
- 承担什么角色
- 为什么值得关注
- 证据等级

## 这个方向谁在做

- 作者：
  - 姓名 / 当前机构
  - 相关问题
  - 代表论文
  - 证据等级

- 实验室：
  - 实验室 / PI
  - 近期连续工作
  - 为什么值得跟踪

- 机构：
  - 机构名
  - 在这个方向的主要切口
  - 代表组或代表论文

## 值得长期跟踪的组

- 组 / 机构
- 跟踪理由
- 建议继续看的论文或项目

## 机构观察

- 方法源头型机构：
- 应用落地型机构：
- 兼顾方法与应用的机构：

## 不确定性和边界

- 哪些是直接证据
- 哪些是推断
- 哪些相关但没有纳入主名单
```

## 搜索时的最小检查清单

1. 先确认目标论文真正属于哪个子方向
2. 先核作者和机构，再扩展到相关组
3. 每个结论至少挂一条证据
4. 优先保留“持续在做”的人和机构
5. 输出里明确写出不确定性
