# 掌握度 Rubric

## 四档掌握度

- `不会`
  - 不能复述核心概念
  - 只能重复关键词

- `模糊`
  - 能复述部分内容
  - 但概念边界、条件或逻辑关系不稳

- `会复述`
  - 能用自己的话解释
  - 能回答基本判断题
  - 但还不够会迁移

- `会迁移`
  - 能解释
  - 能判断
  - 能把它映射到当前研究问题或相邻任务

## 默认小测结构

1. `复述题`
   - 检查是否真正理解而不是背原文
2. `判断题`
   - 检查是否混淆关键概念
3. `迁移题`
   - 检查是否能把知识用到当前方向

## 使用原则

- 小测应短，不要超过 3 题
- 每题都要能指向后续补救动作
- 评分后要给出“下次该补什么”
